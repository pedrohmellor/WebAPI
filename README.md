**WebAPI – CRUD de Clientes**

Este é um projeto de API REST simples para gerenciamento de clientes, desenvolvido com **Node.js**, **Express**, **MySQL** e **dotenv**.  
A API permite cadastrar, editar, excluir e buscar clientes no banco de dados.

**Tecnologias Utilizadas**
- Node.js
- Express
- MySQL2 (Promise API)
- dotenv
- Nodemon (para desenvolvimento)

**Instalação - Clonar o repositório**
bash
git clone https://github.com/pedrohmellor/WebAPI.git
cd WebAPI
